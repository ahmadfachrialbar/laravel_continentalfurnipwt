## ğŸ›  **Instalasi dan Menjalankan Project**

### 1. Clone Repository
```bash
git clone https://github.com/ahmadfachrialbar/laravel_continentalfurnipwt.git
cd laravel_continentalfurnipwt
```
### 2. Install Dependensi
```bash
composer install
npm install && npm run dev
```
### 3. Salin .env
```bash
cp .env.example .env
```
### 4. Migrasi Database 
```bash
php artisan migrate 
```
### 5. jalankan Server
```bash
php artisan serve
Akses di http://localhost:8000
npm run dev
```
